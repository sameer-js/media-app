# media-app

Application to simulate a typical social media app. Also contains user authentication.\
Created using MERN stack (MongoDB for database, Express and NodeJS for backend, React, MUI and Redux Toolkit for frontend).

The frontend of this project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts

In the project directory, you can run:

```
cd server
npm install
(for backend dependencies)


cd client
npm install
(for frontend dependencies)
```

### `npm run start`

In 'client' directory, runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `nodemon index.js`

In 'server' directory, runs the server.\
Server is on [http://localhost:3001](http://localhost:3001).

### `npm test`

Launches the test runner in the interactive watch mode.\
See the section about [running tests](https://facebook.github.io/create-react-app/docs/running-tests) for more information.
